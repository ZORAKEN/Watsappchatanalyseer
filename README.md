# 💬 WhatsApp Chat Analyzer

A modern **WhatsApp Chat Analyzer** built with **Python** and **Streamlit** that provides detailed insights into exported WhatsApp chats. Upload your chat export and instantly visualize user activity, messaging trends, word usage, emojis, links, and more through an interactive dashboard.

---

##  Features

- 📊 Interactive dashboard built with Streamlit
- 💬 Total messages, words, media, and shared links
- 📈 Monthly and daily messaging timeline
- 📅 Weekly and monthly activity analysis
- 🔥 Weekly activity heatmap
- 👥 Most active users in group chats
- ☁️ Word Cloud generation
- 📝 Most frequently used words
- 😂 Emoji usage analysis
- 📱 Clean and responsive WhatsApp-inspired UI
